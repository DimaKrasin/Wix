Программа написана на java 8.
Для запуска через консоль Вам необходим скачать ConsoleProgramme.jar 
Путь: ConsoleProgramme/out/artifacts/ConsoleProgramme_jar


Linux
Для запуска из консоли перейдите в директорию с ConsoleProgramme.jar и пропишите:
java -jar ConsoleProgramme.jar 
или 
java -jar /home/user/ConsoleProgramme.jar 

Windows
Для запуска из консоли перейдите в директорию с ConsoleProgramme.jar и пропишите:
java -jar ConsoleProgramme.jar 
или 
java -jar C:\Users\user\ConsoleProgramme.jar

Важно что бы консоль поддержовала UTF-8
(Для Windows пропишите в консоли "chcp 65001")

Так же в программу добален англиський алфавит по аналогу с гречиским
для удобства использования
Пример англиського алфавита: a-1 b-2 c-3 d-4 .... j-9
